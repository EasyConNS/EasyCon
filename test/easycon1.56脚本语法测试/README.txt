# 1.56愈发完善的全新脚本引擎给伊机脚本带来丰富的执行功能
# 其中有几大新功能下面详细说明

# 1.支持完整的运算表达式语法，不再局限于过去+=, *=等几个赋值运算
# $a = $b + $c * $d
# 完整的四则运算，可以准确的判断运算优先级(先乘除后加减)
# 同时比较表达式也有基本的支持
# $a = 1 > 2(变量$a的类型是布尔型，值是False)
# 布尔型的变量同样可以用在IF表达式中
# IF $a # 如果$a为True则继续执行
# ...
# ELIF 2==3
# ...
# ENDIF

# 2.函数语法支持参数类型标注和返回值
# 经过重构后的脚本引擎支持了完善的函数解析，现在函数不再只是简单的跳转指令
# 每个函数现在都是一个独立完整的脚本执行块，函数中定义的变量只在函数体内生效
# 我们把每个函数体称作一个作用域
# 主脚本中除了函数体语句，其他语句现在属于公共作用域
# 公共作用域的变量可以在函数体中使用
# 函数体中对公共作用域变量的赋值也会造成影响
# $GLOBAL = 0
# FUNC test
# $GLOBAL = 1 # 赋值影响公共变量
# $LOCAL = 0 # LOCAL只在函数作用域
# PRINT $GLOBAL # 输出1
# PRINT $LOCAL # 输出0
# ENDFUNC
# CALL test # 调用test函数
# # PRINT $LOCAL # 错误，找不到变量$LOCAL
# PRINT $GLOBAL # 输出1

# 3. 导入函数功能，现在可以将函数定义写在不同文件中了
# 自从支持函数写法后，伊机脚本功能愈发复杂，能实现的功能越来越多，单文件行数轻易破千
# 将函数分在不同文件中编写有助于合理分割脚本功能模块
# IMPORT "MATH.ecs" # 导入主脚本文件所在目录下的lib/子目录中的MATH.ecs脚本
# $sum = add(1, 2)
# PRINT $sum # 输出3

# 4.搜图标签优先加载脚本文件所在目录下的ImgLabel/子目录中的标签
# 随着脚本越来复杂，搜图标签越做越多难以分类，临时加了这个实验功能
# 目前还有很多问题，后期会持续迭代
# 同名文件将优先加载本地目录中的标签，且伊机公共目录标签会被隐藏
# bug: 保存标签时还是会默认保存到伊机公共目录
